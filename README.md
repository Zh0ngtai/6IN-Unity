#UI
> - 타임 게이지 구현
> - 아이템 획득에 따라 타임 게이지 상승
> - 게임 종료 시 기록 및 최고 기록 저장

#기획의도
> - 게임의 스토리 : 길을 가다 싱크홀에 빠져 벽돌을 클리어하고 지상으로 돌아가기
> - 게임의 컨셉에 맞는 아이템들 출현
> - 생존에는 물이 가장 중요하기 때문에 음료 아이템을 먹게 되면 시간 연장 > 게임 플레이의 가장 중요한 아이템
> - 단계별로 배경이 점점 지상과 가까워짐
> - 동굴이라는 컨셉에 맞는 장애물(박쥐) 등장
